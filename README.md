# Chatbot on Android
### 1. Giới thiệu
Là một chatbot tự động chạy trên HĐH android dựa trên API của Simsimi. Có chức năng trả lời những câu đối thoại mà người dùng đưa ra.
### 2. Yêu cầu cấu hình
#### Về ứng dụng (app)
* Android 5.0 trở lên
#### Về source code
* JDK 1.8.0 trở lên
* Android SDK 11.0 trở lên
